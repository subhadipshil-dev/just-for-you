# 💌 Surprise Page Generator

A fun and interactive web app that lets users create a **personalized surprise webpage** for someone special and share it via a **unique private link**.

This project combines **playful interaction, customization, and temporary content sharing** to create a memorable experience.

---

## ✨ Features

### 🎯 Core Features

* Generate a **unique shareable link**
* Personalized page with:

  * Receiver name
  * Optional sender name
  * Custom message
* **Preview before generating link**
* **24-hour auto-expiry** for all data

---

**🎨 Customization**
* Default Mode (quick setup)
* Advanced Mode:

  * Custom message
  * Image upload (auto-deleted after 24h)
  * Theme selection (optional)

---

**😄 Fun Interaction (Highlight Feature)**
* Interactive **YES / NO buttons**
* On clicking **NO**:

  * Cute/funny GIF appears
  * Message changes dynamically
  * YES button gradually grows bigger 😆
* Playful experience designed to be **fun, not forced**

---

**💖 Final Experience
**
* Clicking **YES** triggers:

  * Celebration animation 🎉
  * Positive message
  * Optional sender reveal

---

**🔐 Privacy & Safety**

* No login required
* Sender name is optional (can stay anonymous)
* Images and data are **automatically deleted after 24 hours**
* No permanent storage of personal content

---

**⚙️ Tech Stack**

**Frontend:**

* React / Next.js
* Tailwind CSS

**Backend:**

* Node.js + Express *(or Firebase)*

**Database:**

* MongoDB / Firestore

**Storage:**

* Cloudinary / Firebase Storage (for temporary images)

---

**🚀 How It Works**

1. Create a page (Default / Advanced)
2. Preview your page
3. Generate a unique link
4. Share it with someone special 💌
5. They open the link and experience the interactive page
